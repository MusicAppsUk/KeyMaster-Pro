# KeyMaster PRO — Music & Repertoire Research Scaffold

Status: **research scaffold only** — no repertoire is imported into the Course yet.
This defines how music enters KeyMaster legally and pedagogically.

## Hard rules (non-negotiable)

- Do **not** copy modern copyrighted editions (engravings/typesetting are themselves
  copyrighted even when the underlying work is public domain — use the *notes*, never
  a scanned edition).
- Do **not** copy ABRSM / Trinity / LCME / RCM syllabus content or ordering.
- Do **not** copy Piano Marvel / Skoove / Simply Piano / Flowkey material, sequencing,
  or expression.
- Verify public-domain / licence status for **every** source before any use.
- Document provenance (composer, work, source, licence, date checked) for everything.
- **Prefer original KeyMaster studies first.** Real repertoire is introduced gradually
  and only when level-appropriate.

## Legal source pool (public domain / open licence — verify each time)

- **IMSLP / Petrucci** — public-domain scores; check each file's tag (some user
  editions carry their own licence; prefer "Public Domain" tagged originals).
- **Mutopia Project** — re-engraved public-domain scores, freely distributable;
  per-piece licence stated (often PD or CC). Provides MusicXML/LilyPond source.
- **Musopen** — public-domain scores and recordings.
- **OpenScore (MuseScore)** — public-domain works re-typeset under CC0; MusicXML.

For each candidate, record: `{ composer, work, movement, source, source_url,
licence, date_verified, notes }` before it touches the build.

## Composer pool (beginner-appropriate, public domain)

Bach (e.g. Notebook for Anna Magdalena minuets), Mozart (early minuets), Beethoven
(simple themes), Clementi (Sonatinas Op.36 — *advanced beginner*), Burgmüller
(Op.100 — early-intermediate), Beyer (Op.101 — true beginner method), Czerny
(simple studies), Schumann (Album for the Young Op.68 — *mostly intermediate*).

> Caution: much "famous" beginner repertoire (Clementi sonatinas, Für Elise,
> Burgmüller) is **harder than a true beginner**. Do not place it in Foundation
> or early Stage 1.

## The KeyMaster approach: original studies inspired by public-domain principles

For beginners we **author original short studies** that teach the same *principles*
as classical models, without copying any specific piece. Examples of the brief:

- "Bach-like stepwise pattern" — a calm, even, mostly-stepwise line in C, hands
  separate; teaches finger independence and legato.
- "Mozart-like question and answer" — a 2-bar rising phrase answered by a 2-bar
  falling phrase; teaches phrase shape and cadence feel.
- "Clementi-style five-finger study" — a five-finger-position figure with a tonic
  close; teaches evenness and a simple I–V–I feel.

Each KeyMaster study is original expression (our own notes), provenance =
"Original KeyMaster study, inspired by <principle>", licence = owned by project.

## Introduction ladder (how real music appears)

1. Foundation — no repertoire; single notes, landmarks, pulse.
2. Stage 1 (Making Music) — original micro-studies (2–4 bars), call-and-response.
3. Stage 2 (Reading & Playing) — original reading studies + the *simplest* genuine
   public-domain fragments (e.g. a 4-bar minuet opening), provenance documented.
4. Stage 3+ — short complete public-domain pieces, level-checked, gradually.

Never imply a beginner will play advanced concerto/sonata material early.
